# Student Performance Prediction System

## Project Overview
This project predicts student performance using Machine Learning.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

## Features
- Predict student marks
- Grade calculation
- Web interface using Flask

## Run Project

### Install dependencies

pip install -r requirements.txt

### Train Model

python model.py

### Run Flask App

python app.py
